Building a Machine Learning

Determining and testing various machine learning model as well as feature engineering

Determining which machine learning model to use.

Gradient Boost Models

This is the best model to use when analyzing customer churn prediction.

  • Captures patterns without needed to manually add a lot of feature interactions

    • Since churn is influenced by many non-linear, subtle patterns, GBM modesl can handle complex relationships automatically.

  • Built-in Feature importance

    • This is extremely useful to help identify which features are most impactful for customer churn.

  • Robus to missing and categorical data

    • Handles missing values well and it reduces data pre-processing.

  • High Acccuracy, Low Bias

    • This minimizes variance and bias which overall helps reduce overfitting.

    Different Types of GBM

  • XGBoost (Extreme Gradient Boosting)
  • Fast
  • Accurate
  • Limitation: Categorical features need one-hot encoding
  • LightGBM (by Microsoft)
  • Faster than XGBoost
  • More memory-efficient than XGBoost
  • Supports categoical features natively
  • Limitation: Can be sensitive to small dataasets or overfitting if not tuned.
  • CatBoost (by Yandex)
  • Best at dealing with categorical variables without preprocessing.
  • Slover than LightGBM but has better accuracy
  • Less sensitive to overfitting
